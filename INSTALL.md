CLI Installation (Local)

macOS / Linux

1. Build binaries:
   ```
   pnpm run build:cli
   ```
2. Install to PATH:
   ```
   ./scripts/install.sh
   ```

Windows (PowerShell)

1. Build binaries:
   ```
   pnpm run build:cli
   ```
2. Install to PATH:
   ```
   .\scripts\install.ps1
   ```

After installation:

- `openokapi` - general command
- `openokapi version` - show version
