# CLI-Installation (Lokal)

## macOS / Linux

1. Binärdateien erstellen:
   ```
   pnpm run build:cli
   ```
2. In PATH installieren:
   ```
   ./scripts/install.sh
   ```

## Windows (PowerShell)

1. Binärdateien erstellen:
   ```
   pnpm run build:cli
   ```
2. In PATH installieren:
   ```
   .\scripts\install.ps1
   ```

## Nach der Installation

Sie können die folgenden Befehle verwenden:

- `openokapi` – Hauptbefehl
- `openokapi version` – Version überprüfen
