## Meldung einer Sicherheitslücke

Verwenden Sie diesen Abschnitt, um Personen mitzuteilen, wie sie eine Sicherheitslücke melden können.

Teilen Sie ihnen mit, wohin sie sich wenden sollen, wie oft sie mit einer Aktualisierung zu einer gemeldeten Sicherheitslücke rechnen können, was sie erwartet, wenn die Sicherheitslücke akzeptiert oder abgelehnt wird usw.
