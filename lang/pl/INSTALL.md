# Instalacja CLI (Lokalna)

## macOS / Linux

1. Zbuduj binaria:
   ```
   pnpm run build:cli
   ```
2. Zainstaluj do PATH:
   ```
   ./scripts/install.sh
   ```

## Windows (PowerShell)

1. Zbuduj binaria:
   ```
   pnpm run build:cli
   ```
2. Zainstaluj do PATH:
   ```
   .\scripts\install.ps1
   ```

## Po Instalacji

Możesz użyć następujących poleceń:

- `openokapi` – główne polecenie
- `openokapi version` – sprawdź wersję
